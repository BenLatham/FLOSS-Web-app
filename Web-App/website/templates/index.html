FLOSS Agricultural Simulation is a: Farm Level, Open Source Simultaion

FLOSS is a farm simulation, aimed at improving understanding of farm systems, and in particular how climate impacts individual farms. This app includes three elements: a wiki documenting the FLOSS code (Documentation), an interface allowing users to enter the model imputs and run a simulation of a particular farm(Simulation), and a user guide for this web interface (User Guide). Please be aware that the model is still under development, and is presented here for demonstration purposes only.

To run the simulation choose "Simulation"

To find out more about the simulation including it's aims, chose "Documentation"

For help on how to run the simulation chose "User Guide"
