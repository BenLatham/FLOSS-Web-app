Future ready:

Browsers are constantly updating and changing – and the last thing you want is for someone to visit your site on a newly updated browser, only to find it doesn't display correctly.

I use the Bootstrap framework, an open-source framework used by many leading sites. The wide community and active development of this framework mean my sites are built from a solid and well tested base. On top of this I use tools to verify the html on my sites, to ensure they comply with the html5 standard – the latest standards from the W3C, the leading international standards organization for the web.

Working from this solid base and complying with these standards gives me confidence that my sites will remain well supported long into the future. In addition my sites are mobile friendly, which is crucial to meet the needs of current and future site users.

If you want to find out more about the html5 standard see: https://www.w3.org/TR/html5/

If you would like to find out more about Bootstrap see: http://getbootstrap.com/
